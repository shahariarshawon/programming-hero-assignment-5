Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM in JavaScript. Such as getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), getAttribute(), childNodes(), parentNode().

Question-2: What is the difference between innerHTML, innerText and textContent?

innerHTML, innerText, textContent each of these behaves differently in DOM of JavaScript. innerHTML shows the whole codes including comments. innerText shows only the text written in a tag. It contains also the css style. textContent shows also shows only the text but does not include css style.>
